Welcome to Backline Boost!

To install: drag the Backline Boost icon onto the Applications folder, then launch it from Applications.

Backline Boost is signed and notarized by Apple. If macOS asks about opening an app downloaded from the internet, click Open.


WHAT'S NEW in 2.2.0 (since 2.1.0)

* A real Mac app install — Backline Boost is now Developer ID-signed and notarized by Apple. No Terminal commands, no security workarounds: drag, launch, play. (The Applications shortcut in this disk image also works properly again; in 2.1.0 it could fail to install.)

* Drums mode — play the isolated drum track by itself. Every source picker now offers Original, Drum Boost, Drumless, and Drums: in the Player, as a playlist default source, and in the mini-player cycle.

* Open in Player from anywhere — library and playlist rows now have a waveform button that opens the track's Player view directly (playlist rows previously had no way to get there).

* Re-render on demand — Ready tracks offer a Re-render action, for after you change the renders folder or render quality in Settings.

* Steadier practice sessions — the app is now a single window; a render finishing in the background no longer interrupts the song you're playing (or have paused); drum-boost changes survive seeking; deleting a track no longer makes Next silently skip a song; and adding several songs to a playlist keeps the order you picked them in.

* Clearer when something's wrong — if playback can't start, Backline Boost tells you why instead of staying silent; a track whose imported file was removed shows "Source missing" while its rendered versions stay playable; and if rendered files go missing, re-rendering starts immediately — even an unplugged renders drive is handled gracefully.

* Faster and lighter — album artwork loads in the background for smoother library scrolling, importing many songs at once no longer repeats loudness analysis work, and failed imports or renders clean up their files instead of leaving them behind.
